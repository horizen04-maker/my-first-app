# アプリ名
DNS（Disaster Nursing Support）

---

## ■ 目的
・国や地域、属性、発災後フェーズ、災害種別等をパラメーター設定し、動画やシミュレーター等、誰でも簡単に学習効率を高め興味を持って取り組めるツール
・ガイドラインやプロトコルを必要に応じて活用し、根拠ある知識・活動につなげる

---

## ■ 対象ユーザー
・初学者
・医療従事者（災害看護のコンピテンシーの段階別）
・学生

---

## ■ 機能一覧
・条件設定（国 / 災害種別 / フェーズ / 対象など）
・学習コンテンツ一覧表示
・コンテンツ詳細表示（動画 / テキスト / シミュレーション）
・シミュレーション実行（簡易）
・入力内容の保存（ローカルストレージ）
・履歴表示

---

## ■ 画面一覧
・トップページ
・条件設定画面（最重要）
・一覧画面
・詳細画面
・シミュレーション画面
・履歴画面

---

## ■ データ設計（超重要）

・学習条件
　- country（国）
　- disaster_type（災害種別）
　- phase（発災フェーズ）
　- target（対象者）

・コンテンツ
　- id
　- title
　- type（動画 / テキスト / シミュレーション）
　- url
　- description

・履歴
　- id
　- 条件
　- 閲覧コンテンツ
　- timestamp

---

## ■ セキュリティ方針（最低限）
・ログインなし
・個人情報を扱わない
・入力値の検証をする（必須）
・外部リンク（動画URL等）の検証
・プロンプトインジェクション対策

---

## ■ データ取り扱いポリシー
・入力データはユーザーの端末内（ローカルストレージ）にのみ保存される
・外部サーバーへ送信しない
・機械学習や外部AIの学習データとして利用しない
・個人情報や患者情報の入力を推奨しない
・医療情報・個人情報の入力は禁止（注意喚起表示を行う）

---

## ■ 技術方針（ざっくりでOK）
・フロント：HTML / CSS / JavaScript
・バックエンド：なし（ローカルストレージ使用）
・公開：GitHub Pages
